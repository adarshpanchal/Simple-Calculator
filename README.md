# 🧮 Calculator Web App

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Basic arithmetic operations (+, -, *, /)
* Backspace functionality (⌫)
* Error handling
* Clean UI with custom styling

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (DOM Manipulation)
